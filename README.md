# JavaScript Math and Calculation Functions

## Description

This project contains a collection of JavaScript functions designed to solve mathematical and geometric problems. The exercises focus on calculations, function creation, parameter handling, DOM manipulation, and algorithm development.

The project includes utilities for BMI calculation, factorial computation, currency conversion, area and perimeter calculations, and multiplication table generation.

## Features

### 1. BMI Calculator

* Calculates a person's Body Mass Index (BMI).
* Uses height and weight as parameters.
* Returns the calculated BMI value.

### 2. Factorial Calculator

* Calculates the factorial of a given number.
* Uses iterative logic with a loop.

### 3. Currency Converter

* Converts US Dollars (USD) to Brazilian Reais (BRL).
* Uses a fixed exchange rate.

### 4. Rectangle Area and Perimeter Calculator

* Calculates the area and perimeter of a rectangular room.
* Displays the result on the webpage.

### 5. Circle Area Calculator

* Calculates the area of a circular room.
* Uses the radius as a parameter.
* Displays the result on the webpage.

### 6. Multiplication Table Generator

* Generates and displays the multiplication table of a given number.
* Uses loops to perform the calculations.
